# Common Types

Common types in Verse include fundamental data types that are frequently used in programming.

## Basic Types

- `int` - Integer numbers
- `float` - Floating-point numbers
- `string` - Text strings
- `logic` - Boolean values (true/false)
- `char` - Single characters

## Container Types

See [Container Types](container_types.md) for arrays, maps, and other collections.

## Type System

Verse has a strong, static type system with type inference capabilities.
